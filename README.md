# To-do list

## Descriprion

Local to-do list api with web interface.

## Functional

- Add, delete tasks or change it status
- Data saves locally

## Installation and launch

### Requirements

- Install [Go 1.24.3+](https://go.dev/doc/install) or [Docker](https://docs.docker.com/get-started/get-docker)

### Quick start

1. **Clone repository**

```bash
git clone https://github.com/HeDoBoJlEH/To-do-list.git
cd todo_list_api
```

2. **Run**

- **Via Golang**:

```bash
go run app/cmd/main.go
```

- **Via Docker**:

```bash
docker-compose up
```

Note: you should run Docker Desktop
